推荐使用

本篇适合作为正式长文训练。它有大量神话例子，但中心很清楚：读神话不是记奇怪故事，而是借神话照见人的处境、欲望、恐惧、选择和命运。

八轴观察重点

主看 A4/A6/A7
  • A3：能否定位作者偏爱原因、神的复杂性、现代对应、镜子含义。
  • A4：能否把“生活气息 -> 神的冷酷 -> 现代世界 -> 心灵镜子”压成链条。
  • A6：能否区分生活气息证据、现代对应证据和人生问题证据。
  • A7：能否解释古老神话与今天生活的关系，而不是只写有趣。
  • A8：能否说出“先找观点，再给例子分类”的策略。

原文证据池

批改先看这里
  1. 作者偏爱古希腊神话，因为它们狂野奇特、富有生活气息。
  2. 古希腊人眼中神无处不在，参与人的一思一行。
  3. 神并不总是善的，可能冷酷、暴力、任性。
  4. 这些神代表暴力、不可预知、不公正的世界。
  5. 疾病、战争、洪水、大火、法律与正义相悖等问题今天仍存在。
  6. 伊卡洛斯、法厄同、那喀索斯等可对应今天的灾难、技术和欲望。
  7. 神话是一面镜子，因为我们在其中照见自己的样子。
  8. 史诗讨论生、死、爱、美、命运、身份、战争等重大问题。

批改解读

答案要点 / 支架 / 回流

第 1 题：信息定位与结构识别

A2 / A3 · 3分

答案要点：作者偏爱古希腊神话，因为它们狂野奇特、富有生活气息；神并不总是善的，可能任性、冷酷、暴力；现代世界仍有灾难、不公、欲望和不安；镜子照见人的恐惧、欲望、悲伤、选择、命运和道德两难。

常见低阶答案：只写“神话很有趣”；把“镜子”理解成照脸；只列神话名字。

画像回收：能定位但不能概括，回到 A4；文体误判为故事合集，回到 A2。

第 2 题：篇章主线整合

A4 · 3分

参考链条：古希腊神话有生活气息 -> 古希腊人相信神无处不在 -> 但神并不总是善的 -> 他们代表暴力、不可预知、不公正的世界 -> 现代世界仍有灾难、不公、欲望和不安 -> 神话像镜子，让我们照见自己。

助学者提示：这题不是问有哪些神话人物，而是问作者观点怎么一步步推进。

画像回收：不能从例子压成链条，回到 A4。

第 3 题：核心证据题

A6 · 3分

参考答案：最能支持“镜子”的证据是 C 和 D。C 说明神话中的灾难、不公和人的困境今天仍存在；D 说明史诗讨论人类共同问题。B 也有支持作用。A 主要说明神话有生活气息，不能直接证明“照见我们自己”。

常见低阶答案：选 A，因为例子多、画面感强；只抄中心句，不解释证据关系。

画像回收：只看例子多不看证明力，回到 A6。

第 4 题：评价迁移题

A7 / A6 · 2分

参考方向：不同意。原文说疾病、战争、洪水、大火、法律与正义相悖等问题今天仍存在；伊卡洛斯、法厄同、那喀索斯等还能对应今天的气候灾难、人工智能、自拍和不安。因此古希腊神话能照见现代人的处境。

助学者提示：不要只说“有关系”，要说清是哪一种关系：灾难相似、欲望相似、选择相似，还是人的不安相似。

画像回收：迁移空泛回到 A7；有证据但解释不出作用，回到 A6。

第 5 题：A8 复盘题

A8 · 1分

满分要点：中心句可写“神话是一面镜子，因为我们在其中照见了自己的样子”；例子可分成生活气息类、神的冷酷类、现代对应类、人生问题类；策略是先找观点句，再问每个例子证明哪个观点。

助学者提示：例子很多时，不能先背名字。先找观点，再给例子贴标签。
