# 范润然自选素材第 7 篇阅读题：《打树花》

> 版本：2026-06-12  
> 素材来源：用户提供 Obsidian 文档《冯骥才-打树花》  
> 使用对象：范润然，五年级  
> 训练定位：训练她读懂“民俗场景散文”的动作顺序、场面描写、人物形象和文化价值，不只停留在“很壮观、很好看”的感受层。

---

## 1. 素材可用度结论

综合结论：**推荐使用。**

理由：

1. 文章画面感强，动作链清楚：看场、汉子登场、祭炉、开炉、舀铁水、泼墙、铁花飞溅、王全上阵、民俗技艺解释。
2. 可训练 A4：学生需要把热闹场面压缩成“打树花的过程”和“文章行文顺序”。
3. 可训练 A5：王全不是直接被评价，而是通过动作、装束、年龄、身份和坚持回乡表演被写出“勇敢、技艺高、热爱民俗”。
4. 可训练 A6：大量比喻、夸张和动作细节都可以变成证据，考查“描写怎样证明壮观和勇气”。
5. 可训练 A7：文章后半部分补充铜、铝、柳木勺等民俗技艺，适合引导学生评价民俗的美、险、智慧与传承。

本篇主目标轴：

- A4 篇章整合与概括
- A5 推论与人物理解
- A6 证据整合与论证支持

本篇辅助观察轴：

- A2 文本类型与文体识别
- A3 信息定位
- A7 评价判断与迁移
- A8 元认知监控与策略

不建议强行训练的轴：

- A1 不作为主训练。少量词语如“灼灼、依稀、祭炉、枪林弹雨、避火”可做理解辅助。

阅读难度判断：**合适偏难。**

预计最容易暴露的卡点：

- 只写“很壮观”，不能按顺序讲清打树花过程。
- 能找出比喻句，但说不出它表现了什么。
- 评价王全时只写“勇敢”，没有用原文动作和装束作证据。
- 读到后半部分的民俗知识时，觉得与前文场面无关，不能看出它在解释“为什么这项民俗既美又有技艺”。

是否适合进入本轮训练：**适合。**

## 4. 学生训练题

### 第 1 题：过程梳理题

请根据原文，把“打树花”的现场过程按顺序补完整。

```text
夜色中的广场和炉火准备好 -> 几个大汉登场，穿着__________ -> 他们先__________ -> 开炉放铁水，两个大汉把铁水抬到广场中央 -> 一个大汉用长柄勺__________ -> 铁水泼到城墙上，形成__________ -> 王全上阵，铁花像__________ -> 作者最后补充打树花的__________和__________
```

### 第 2 题：场面描写赏析

请从下面两组句子中任选两处，说明它们写出了打树花怎样的特点。

A. “就在这一瞬，好似天崩地裂。”  
B. “金红的铁水泼击墙面，四外飞溅。”  
C. “如同细密的光雨闪闪烁烁由天而降。”  
D. “铁花升腾时如在头上张开一棵辉煌又奇幻的大树。”

建议作答：

```text
我选择____和____。
____运用了__________，写出了__________。
____运用了__________，写出了__________。
这两处共同表现了打树花__________的特点。
```

### 第 3 题：人物形象题

文章为什么说王全像“枪林弹雨中一个无畏的勇士”？请至少用三处原文证据说明王全是一个怎样的人。

要求：

1. 先概括王全的特点；
2. 至少写三处证据；
3. 说明每处证据分别证明了什么。

### 第 4 题：民俗价值评价题

文章后半部分写了打树花的来历、花色变化、柳木勺泡水等内容。有人说：“这些知识和前面的壮观场面关系不大，可以删掉。”你同意吗？请根据原文回答。

要求：

1. 表明观点；
2. 至少用两处后半部分的原文证据；
3. 说明这些内容怎样帮助我们理解打树花这项民俗。

### 第 5 题：A8 复盘题，不计主分

这篇文章很容易让人只写“壮观、好看”。你做题时怎样避免空泛？

## 5. 出题人与助学者用答案要点

### 第 1 题

对应能力轴：A3 / A4

参考答案：夜色中的广场和炉火准备好；几个大汉登场，穿草帽、老羊皮袄、腰扎粗绳、脚遮布帘；他们先点香、烧黄纸、祭炉；开炉放铁水；大汉用长柄勺舀铁水、甩腰抡臂、泼到城墙上；铁水形成光雨、火炮、铁花；王全上阵，铁花像一棵辉煌又奇幻的大树；作者最后补充打树花的来源、花色变化、柳木勺使用等民俗技艺。

常见低阶答案：只写铁水泼墙；漏掉祭炉和后半部分技艺解释；顺序混乱。

助学者提示：让她先给段落编号，再写“准备 -> 仪式 -> 表演 -> 高潮 -> 民俗解释”。

评分点：3 分。顺序完整 1 分，关键动作准确 1 分，能纳入后半部分民俗解释 1 分。

画像回收：顺序乱或只记局部，回收到 A4；事实找不到，回到 A3。

### 第 2 题

对应能力轴：A6

参考方向：

- A “天崩地裂”用夸张写出铁水泼墙瞬间的震撼、声势和危险感。
- B “金红”“四外飞溅”从颜色和动态写出铁水灿烂、猛烈。
- C “细密的光雨”用比喻写出铁花落下时密集、闪烁、优美。
- D “辉煌又奇幻的大树”照应题目“树花”，写出铁花升腾的形状、美感和神奇。

常见低阶答案：只写“很好看”；能说修辞但说不出特点；把所有句子都解释成“壮观”，没有区分。

助学者提示：每句都问两个问题：作者用了什么词？这个词让你看到什么画面？

评分点：3 分。两处选择准确 1 分，能解释修辞或关键词 1 分，能概括共同特点 1 分。

画像回收：只抒情不解释证据，回到 A6。

### 第 3 题

对应能力轴：A5 / A6

参考答案方向：王全是一个勇敢、技艺高超、热爱并坚持传承民俗的人。证据包括：他是北官堡打树花的“武状元”，说明技艺出众；他61岁仍年年回来给乡亲们表演，说明坚持和热爱；他泼铁水时“又快又猛又有韵律”，铁花飞得高、散得广，说明经验丰富、功夫好；别人戴有檐草帽，他戴无檐小毡帽，更显出勇气；铁水温度高达1300摄氏度，表演本身很危险，也衬托他的胆量。

常见低阶答案：只写“勇敢”；只引用一句“武状元”；没有说明证据和人物特点之间的关系。

助学者提示：人物形象题不要只贴标签，要用“特点 + 证据 + 说明”。

评分点：3 分。人物特点准确 1 分，三处证据 1 分，证据解释到位 1 分。

画像回收：有人物标签无证据，回到 A6；不能从细节推人物，回到 A5。

### 第 4 题

对应能力轴：A7 / A6

参考答案方向：不同意删掉。后半部分说明打树花不是单纯热闹的表演，而是经过漫长岁月形成的民俗。比如铁水、铜水、铝水打出的“花”颜色不同，人们逐渐加入铜和铝，使树花更美，说明民俗会在传承中发展。柳木勺要泡一天一夜，用七八下就放回木桶，说明民间艺人有应对高温铁水的经验和智慧。铁水温度高达1300摄氏度，也让我们更理解表演者的勇气和技艺。因此这些内容帮助我们理解打树花的文化价值、技术难度和传承意义。

常见低阶答案：只写“不能删，因为很重要”；只复述后半部分知识，没有解释作用；只说“更有趣”，没有连回民俗价值。

助学者提示：问她：后半部分让你更懂了打树花的哪一面？美？险？智慧？传承？

评分点：2 分。观点明确 1 分，能用两处证据说明民俗价值 1 分。

画像回收：评价空泛回到 A7；证据作用解释不足回到 A6。

### 第 5 题

对应能力轴：A8

满分答案要点：能说出先梳理“准备、仪式、表演、高潮、民俗解释”的顺序；圈动作词如舀、甩腰、抡臂、泼、飞溅、升腾；找描写句如天崩地裂、光雨、大树；评价人物或民俗时必须补原文证据。

常见低阶答案：“我会认真读”；“我会多找好词好句”；没有具体策略。

助学者提示：场面描写类文章要按“顺序 + 动作 + 画面 + 中心”读。

评分点：1 分。能说出具体阅读策略即可。

画像回收：回到 A8 的场面描写阅读策略。

## 6. 助学者执行单

本篇训练主目标：A4 场面过程整合；A5 人物形象推断；A6 描写证据解释。

先不给提示时，观察：

1. 她是否能按顺序复述打树花，而不是只记住铁水飞溅。
2. 她是否能从动作、装束、年龄、身份推断王全形象。
3. 她是否能说明比喻、夸张和动作词的作用。
4. 她是否能看出后半部分民俗知识是在解释“为什么这项民俗美、险、有智慧、值得传承”。

如果 A3 卡住，提示：

```text
先找人物、数字和动作：王全、61岁、七八下、1300摄氏度、泡一天一夜。
```

如果 A4 卡住，提示：

```text
按顺序写五步：准备 -> 祭炉 -> 舀铁水泼墙 -> 王全高潮表演 -> 民俗技艺解释。
```

如果 A5 卡住，提示：

```text
人物形象不是凭感觉写，要找装束、动作、别人评价、年龄和坚持。
```

如果 A6 卡住，提示：

```text
每个描写句都要回答：它让我看见什么？它表现了打树花的美、险、壮观，还是人物勇气？
```

如果 A7 卡住，提示：

```text
后半部分不是闲笔，它把“好看”推进到“民俗智慧和传承”。
```

本次可撤掉的支架：

- 如果她能独立完成第 2 题，下次可减少“修辞 + 画面 + 特点”的提示。

本次必须保留的支架：

- “人物特点 + 原文证据 + 证明作用”的支架。

下一篇素材建议：

- 可继续选民俗、非遗、人物叙事类文章，训练“场面描写 -> 人物精神 -> 文化意义”的三级整合。

## 7. 技能规则总结

当原文出现“民俗场景 + 动作链 + 人物特写 + 技艺解释”时，可以生成场面过程整合题、描写赏析题、人物形象证据题和民俗价值评价题。  
该题主要训练 A4、A5、A6。  
应调用粗读找顺序、细读看动作和描写、精读评价人物与文化价值。  
低阶学生容易只写“壮观、好看、勇敢”，但缺少原文证据。  
脚手架应先让学生梳理过程，再让学生圈动作和描写，最后让学生用证据评价人物和民俗。  
评分时重点看顺序是否完整、证据是否贴合、评价是否能回到文本。
