# 范润然自选素材第 3 篇训练题：《为什么读古希腊神话？》

> 版本：2026-06-11  
> 素材来源：用户提供文章《为什么读古希腊神话？》  
> 使用对象：范润然，五年级  
> 训练定位：训练她读懂“例子很多、观点逐层推进”的议论性散文，不停留在神话故事表面。

---

## 1. 素材可用度结论

综合结论：**推荐使用，但需要较强结构支架。**

理由：

1. 文章结构完整，从“我为什么偏爱古希腊神话”写到“为什么神话是一面镜子”，再引向人生问题的思考。
2. 文章原文证据丰富，适合训练 A4 篇章整合、A6 多证据支持、A7 评价迁移。
3. 文章有大量神话人物和文化典故，容易让学生被名字带跑，需要助学者帮她抓住“例子服务观点”的阅读方法。
4. 文章中心很清楚：读古希腊神话不是为了记奇怪故事，而是借神话照见人的处境、欲望、恐惧、选择和命运。

本篇主目标轴：

- A4 篇章整合与概括
- A6 证据整合与论证支持
- A7 评价判断与迁移

本篇辅助观察轴：

- A2 文本类型与文体识别
- A3 信息定位
- A8 元认知监控与策略

不建议强行训练的轴：

- A1 不作为主训练。本文有较多专名和抽象词，但不要把本篇变成希腊神话知识课或词语解释课。
- A5 不作为主训练。人物和神很多，但本篇重点不是分析某一个人物性格，而是理解作者如何用人物例子支撑观点。

阅读难度判断：**偏难但值得用。**

预计最容易暴露的卡点：

- 记住了很多神话名字，但说不清这些例子分别证明什么。
- 能找到“神话是一面镜子”，但解释不出“镜子”照见了什么。
- 迁移题容易泛泛写“神话很有趣、能学知识”，没有回到文本中的“人类共同问题”。

是否适合进入本轮训练：**适合，建议作为第 3 篇正式长文训练材料。**

## 4. 学生训练题

### 第 1 题：信息定位与结构识别

请根据原文回答：

1. 作者一开始说自己为什么偏爱古希腊神话？
2. 作者后来发现古希腊神并不总是怎样的存在？
3. 作者为什么说 3000 多年后，神话仍然和今天有关？
4. 作者说“神话是一面镜子”，这面镜子照见了什么？

### 第 2 题：篇章主线整合

请把文章的论证链补完整：

```text
古希腊神话狂野奇特，也有明媚生活气息
-> 古希腊人相信神 ____________________
-> 但这些神并不总是 ____________________
-> 他们代表了一个 ____________________ 的世界
-> 3000 多年后，现代世界仍然有 ____________________
-> 所以神话像一面镜子，让我们 ____________________
```

### 第 3 题：核心证据题

作者为什么说“神话是一面镜子”？下面四组证据中，哪两组最能支持这个观点？哪一组主要是在说明“生活气息”，不能直接证明“镜子”？请说明理由。

A. 大海咆哮是波塞冬发怒，露珠是母亲悲伤的眼泪，水仙花来自少年迷恋倒影。  
B. 古希腊神并不总是善的，他们任性、冷酷，代表暴力、不可预知、不公正的世界。  
C. 疾病、战争、洪水、大火、法律与正义相悖等问题，今天仍然存在。  
D. 《伊利亚特》和《奥德赛》讨论生、死、爱、美、命运、身份、战争等重大问题。

请这样答：

```text
我认为最能支持“神话是一面镜子”的证据是____和____。
因为____证明了____，____证明了____。
____主要说明神话有生活气息，但不能直接证明“镜子”，因为____。
```

### 第 4 题：评价迁移题

如果有同学说：“古希腊神话太古老了，和我们现在的生活没有关系。”你同意吗？

请根据原文回答，要求：

1. 先表明同意或不同意。
2. 至少用两处原文证据。
3. 说明这些证据怎样证明“古老神话仍然和今天有关”。
4. 不能只写“能增长知识”或“故事很有趣”。

### 第 5 题：A8 复盘题，不计主分

这篇文章例子很多。你做题时是怎么防止自己被神话名字带跑的？

请补全：

```text
我先找出的中心句是：____________________
我把例子分成了几类：____________________
我认为最重要的一类例子是：____________________
因为它能证明：____________________
下次读例子很多的文章，我要先问自己：____________________
```

## 5. 出题人与助学者用答案要点

### 第 1 题

对应能力轴：A2 / A3

原文依据：开头偏爱原因、阿克泰翁段、现代世界段、“神话是一面镜子”段。

满分答案要点：

1. 作者偏爱古希腊神话，因为它们狂野奇特，富有明媚生活气息。
2. 古希腊神并不总是善的存在，可能任性、冷酷、暴力。
3. 因为现代世界仍有疾病、战争、灾难、不公正、欲望和不安等问题，古代神话仍能解释人的处境。
4. “镜子”照见的是人自己的样子，包括恐惧、欲望、悲伤、选择、命运和道德两难。

常见低阶答案：

- 只写“因为神话很有趣”。
- 把“神话是一面镜子”解释成“可以照脸”。
- 只列神话名字，不说明含义。

助学者提示：

```text
先找每一大段在回答什么问题：为什么喜欢？神是什么样？今天为什么还相关？镜子照见什么？
```

评分点：3 分。四问基本准确 3 分；漏掉深层含义扣 1 分；只列例子不概括扣 1 分。

画像回收：若能定位但不能概括，回收到 A4；若文体误判为故事合集，回收到 A2。

### 第 2 题

对应能力轴：A4

参考答案：

```text
古希腊神话狂野奇特，也有明媚生活气息
-> 古希腊人相信神无处不在、参与人间事务
-> 但这些神并不总是善的存在
-> 他们代表了一个暴力、不可预知、并不公正的世界
-> 3000 多年后，现代世界仍然有疾病、战争、灾难、不公正、欲望和不安
-> 所以神话像一面镜子，让我们照见自己的样子
```

常见低阶答案：

- 写成神话人物列表。
- 漏掉“神不总是善的”这一转折。
- 最后只写“神话很有用”，没有写“照见自己”。

助学者提示：

```text
这题不是问有哪些神话人物，而是问作者的观点怎么一步步推进。先抓转折：“但”“3000多年过去了”“因为”。
```

评分点：3 分。生活气息与神参与 1 分，神话世界判断 1 分，现代对应与镜子主旨 1 分。

画像回收：若不能从例子压成链条，回收到 A4。

### 第 3 题

对应能力轴：A6

参考答案：

最能支持“神话是一面镜子”的证据是 C 和 D。C 说明神话中的灾难、不公和人的困境在今天仍然存在，所以神话能照见现代人的处境；D 说明古希腊史诗讨论的是生、死、爱、美、命运、身份、战争等人类共同问题，所以它能照见人的内心和选择。B 也有支持作用，因为它说明神话世界像现实世界一样复杂、不公正。A 主要说明古希腊神话有生活气息，能把自然现象讲成故事，但它不能直接证明“神话照见我们自己”。

常见低阶答案：

- 选 A，因为 A 例子多、画面感强。
- 只抄“神话是一面镜子”，不解释证据关系。
- 把所有例子都当成同等核心证据。

助学者提示：

```text
题干问的是“镜子”，不是问“神话有趣”。能照见今天的人和问题的证据，才更核心。
```

评分点：3 分。选出核心证据 1 分，解释每条证据作用 1 分，能区分生活气息证据与镜子证据 1 分。

画像回收：若只看例子多不看证明力，回收到 A6。

### 第 4 题

对应能力轴：A7，可兼 A6

参考答案方向：

不同意。原文说，3000 多年过去了，疾病、战争、洪水、大火、法律与正义相悖等问题仍然存在，这说明神话里关于灾难和不公的想象仍能帮助我们理解今天的世界。原文还说，伊卡洛斯的坠落、法厄同的翻车像写给今天气候灾难和人工智能的寓言，那喀索斯仍然活在自拍的年轻人心中，这说明古老神话可以对应现代人的欲望、风险和不安。因此，古希腊神话并不是只属于过去，它能让我们借一面异域的镜子照见自己的心灵。

常见低阶答案：

- “不同意，因为神话很好看。”
- “同意，因为古希腊离我们太远。”但忽略原文后半部分。
- 能引用例子，但不解释例子与今天的关系。

助学者提示：

```text
你不能只说“有关系”，要说清是哪一种关系：灾难相似？欲望相似？选择相似？人的不安相似？
```

评分点：2 分。观点明确 1 分，能用两处证据解释“古老与今天的联系” 1 分。

画像回收：若迁移空泛，回收到 A7；若有证据但解释不出作用，回收到 A6。

### 第 5 题

对应能力轴：A8

满分答案要点：

1. 中心句可写“神话是一面镜子，因为我们在其中照见了自己的样子”。
2. 例子分类可写：生活气息类、神的冷酷类、现代对应类、人生问题类。
3. 最重要的一类通常是“现代对应类”或“人生问题类”，因为它们最能证明为什么今天还要读神话。
4. 策略是先找观点句，再问每个例子证明哪个观点。

常见低阶答案：

- “我认真读名字。”
- “我把人物都背下来。”
- 只写最喜欢哪个神话人物，没有复盘阅读方法。

助学者提示：

```text
例子多的文章，不能先背例子。先找观点，再给例子贴标签：它证明生活气息、神的复杂，还是照见今天？
```

评分点：1 分。能说出“先找观点句，再分类例子”的策略即可。

画像回收：回收到 A8 的长文例子管理策略。

## 7. 助学者执行单

本篇训练主目标：A4 把大量例子压成论证链；A6 判断哪些例子真正支持“镜子”；A7 解释古老神话与今天生活的关系。

先不给提示时，先观察：

1. 她是否把文章当成神话故事列表。
2. 她是否能找出“神话是一面镜子”这个中心句。
3. 她能否说明例子证明了什么，而不是只复述例子。

如果 A3 卡住，提示：

```text
先找四个位置：作者为什么喜欢？神有什么复杂性？今天有什么对应？镜子的解释在哪里？
```

如果 A4 卡住，提示：

```text
不要先背人物名字。先用一句话说每段在推进什么观点。
```

如果 A6 卡住，提示：

```text
A 类例子说明神话有趣，C/D 类例子说明神话能照见今天。题干问“镜子”，所以后者更核心。
```

如果 A7 卡住，提示：

```text
你的迁移要回到文中的“大问题”：灾难、不公、欲望、不安、选择、命运。不要只写“增长知识”。
```

如果 A8 复盘说不清，提示：

```text
例子很多时，你怎么处理？先找观点句，再给例子分类，而不是按人物名字一个个记。
```

本次可撤掉的支架：

- 如果她能独立完成第 1 题和第 2 题，下次可减少段落定位提示。

本次必须保留的支架：

- “观点句 -> 例子分类 -> 证据作用”的三步支架。
- “古老神话 -> 现代处境”的迁移边界支架。

下一篇素材建议：

- 可以继续选择“例子多、观点清楚”的文化类文章。
- 但下一篇最好减少专名密度，观察她是否能把本篇的例子分类策略迁移过去。
