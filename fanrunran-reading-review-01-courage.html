推荐使用

本篇适合作为第 1 篇 A4+A6 训练材料，但需要支架。文章结构清楚：高墙故事 -> 妈妈解释勇气 -> 亚里士多德观点 -> 黄金中道与实践智慧 -> 小刚变化 -> 最终升华。

本篇最值得读懂的是：勇气不是不害怕，而是在害怕时仍能做清醒、负责、正确的选择。

原文证据池

批改先看这里
  1. 小明也害怕高墙，但选择陪小刚一起面对。
  2. 小明在攀爬中一直鼓励小刚，一步一步帮助他完成。
  3. 妈妈说勇气不是不害怕，而是即使害怕也面对正确的困难。
  4. 亚里士多德指出不能只把勇气理解为克服怯懦。
  5. 校园霸凌例子区分了怯懦、鲁莽和合理负责的行动。
  6. 实践智慧要求行动前考虑对象、方式和后果。
  7. 小刚几个月后主动再次攀爬，说明勇气可以影响和传递。
  8. 结尾概括真正的勇气是清醒而负责任的选择。

批改解读

答案要点 / 支架 / 回流

第 1 题：信息定位与文本结构

A2 / A3 · 3分

满分要点：先写小明陪小刚攀爬高墙；中间借妈妈和亚里士多德解释勇气；最后用小刚主动再次挑战说明勇气可以传递。

常见低阶答案：只写“小明和小刚爬墙”；漏掉亚里士多德；把最后环境描写当主要内容。

助学者提示：先切三段：故事在哪里？讲道理从哪里开始？最后又回到谁身上？

画像回收：若只能复述故事，回收到 A2/A4 的混合文本结构识别不足。

第 2 题：主线整合

A4 · 3分

参考链条：看到高墙也害怕 -> 选择陪小刚一起面对 -> 听妈妈解释勇气 -> 理解勇气要有理性和分寸 -> 看到小刚后来主动挑战 -> 明白真正的勇气是在害怕时仍做清醒、负责、正确的选择。

常见低阶答案：写成“小明很勇敢，小刚也勇敢”；只写事件，不写理解变化；把“勇气就是不害怕”当成结论。

助学者提示：这题不是问发生了什么，而是问“理解怎么变深”。每个箭头后面都要写一个变化。

画像回收：若能找事件但不能压成变化链，回收到 A4。

第 3 题：核心证据题

A6 · 3分

参考答案：最核心的证据是 B 和 D。B 证明小明不是没有害怕，而是在害怕时仍选择陪伴和鼓励朋友；D 证明真正的勇气不是退缩，也不是冲动，而是根据情况做合理负责的选择。A 只是背景证据，因为它说明两人关系好，但不能直接证明什么是真正的勇气。C 是辅助说明，可以帮助理解“不能简单理解”，但不如 B 和 D 直接证明题干中的“清醒、负责地面对困难”。

常见低阶答案：选 A，因为朋友关系好；选 C 但只说“很有道理”；把所有材料都当核心证据。

助学者提示：先看题干关键词：“不是无所畏惧”“清醒”“负责”。哪条材料直接说明这些词？

画像回收：若只摘材料不解释“证明了什么”，回收到 A6。

第 4 题：评价迁移题

A7 / A6 · 2分

满分方向：做法要具体，例如陪同学准备、鼓励他一步步完成、必要时请老师创造安全支持。要说明这不是退缩，因为没有袖手旁观；也不是鲁莽，因为没有起哄、硬推、责备或逼迫同学。可用小明陪伴小刚、实践智慧考虑对象方式后果、校园霸凌例子中的合理行动作为依据。

常见低阶答案：只写“我要勇敢帮助他”；让同学马上上台，忽略对方害怕；只说“不要怕”，没有文本依据。

助学者提示：你的做法有没有同时避开两边：一边是不管他，一边是硬逼他？文中哪句话能支持这种分寸？

画像回收：迁移只写常识，回收到 A7；有证据但不会解释，回收到 A6。

第 5 题：A8 复盘题

A8 · 1分

满分要点：能说出题干要证明的是“真正勇气的定义和边界”；能解释为什么保留 B/D、为什么删掉 A 或降低 C 的权重；能形成下次策略，如“先看题干关键词，再看证据是否直接证明它”。

常见低阶答案：“我凭感觉选的”；“哪个句子长就选哪个”；“我觉得都可以”。

助学者提示：你不是在选喜欢的句子，而是在选最能证明观点的句子。它能不能回答题干关键词？

画像回收：回收到 A8 的证据选择策略。

助学者执行单

现场支架
  • 如果 A3 卡住：先找三个位置，高墙故事、亚里士多德、小刚后来变化。
  • 如果 A4 卡住：追问“你现在写的是事情顺序，还是理解越来越深的顺序？”
  • 如果 A6 卡住：追问“这条证据能不能直接证明清醒负责？”
  • 如果 A7 卡住：追问“你的做法有没有一边避免退缩，一边避免鲁莽？”
  • 如果 A8 说不清：让她说明删掉 A 的理由，不是因为 A 错，而是证明力不够直接。

下一篇建议：继续选择“故事 + 明确观点 + 至少两个例子”的材料，让 A6 证据判断稳定下来。
