推荐使用

本篇适合作为第 7 篇民俗场景散文训练材料。它既有现场动作顺序，又有王全的人物特写，还在结尾补充民俗技艺和民间智慧，能把“好看”推进到“为什么好看、为什么难、为什么值得传承”。

批改解读

答案要点 / 支架 / 回流

第 1 题：过程梳理题

A3 / A4 · 3分

答案要点：准备炉火和广场；大汉登场，穿草帽、老羊皮袄、腰扎粗绳、脚遮布帘；点香、烧黄纸、祭炉；开炉放铁水；大汉舀铁水、甩腰抡臂、泼到城墙上；铁花形成光雨、火炮、大树；最后补充来源、花色变化和柳木勺等民俗技艺。

常见低阶答案：只写泼铁水；漏掉祭炉；不把后半部分技艺解释纳入文章结构。

助学者提示：按“准备 -> 仪式 -> 表演 -> 高潮 -> 民俗解释”重新画一条线。

画像回收：顺序乱回到 A4；事实漏找回到 A3。

第 2 题：场面描写赏析

A6 · 3分

参考方向：“天崩地裂”用夸张写出铁水泼墙瞬间的震撼、声势和危险感；“金红”“四外飞溅”从颜色和动态写出铁水灿烂、猛烈；“细密的光雨”用比喻写出铁花落下时密集、闪烁、优美；“辉煌又奇幻的大树”照应题目，写出铁花升腾的形状、美感和神奇。

常见低阶答案：只写“很好看”；能说修辞但说不出具体画面；所有句子都只解释为“壮观”。

助学者提示：每句问两步：用了什么词？这个词让你看见什么？

画像回收：只抒情不解释证据，回到 A6。

第 3 题：人物形象题

A5 / A6 · 3分

参考方向：王全勇敢、技艺高超、热爱并坚持传承民俗。证据：他是打树花“武状元”，说明技艺出众；61岁仍年年回来表演，说明坚持和热爱；泼铁水又快又猛又有韵律，说明经验和功夫好；别人戴有檐草帽，他戴无檐小毡帽，衬托勇气；铁水1300摄氏度，表演危险，衬托胆量。

常见低阶答案：只写“勇敢”；只引用“武状元”；证据和特点之间没有解释。

助学者提示：人物形象题用“特点 + 证据 + 说明”。

画像回收：有人物标签无证据，回到 A6；不能从细节推人物，回到 A5。

第 4 题：民俗价值评价题

A7 / A6 · 2分

参考方向：不同意删掉。后半部分说明打树花不是单纯热闹的表演，而是经过漫长岁月形成的民俗。铁水、铜水、铝水花色不同，人们逐渐加入铜和铝，使树花更美，说明民俗会在传承中发展。柳木勺要泡一天一夜，用七八下就放回木桶，说明民间艺人有应对高温铁水的经验和智慧。铁水温度高达1300摄氏度，也让我们更理解表演者的勇气和技艺。

常见低阶答案：只写“不能删，因为重要”；只复述知识，没有解释作用；只说“更有趣”，没有回到民俗价值。

助学者提示：问她：后半部分让你更懂了打树花的哪一面？美？险？智慧？传承？

画像回收：评价空泛回到 A7；证据作用解释不足回到 A6。

第 5 题：A8 复盘题

A8 · 1分

满分要点：能说出先梳理“准备、仪式、表演、高潮、民俗解释”的顺序；圈动作词如舀、甩腰、抡臂、泼、飞溅、升腾；找描写句如天崩地裂、光雨、大树；评价人物或民俗时必须补原文证据。

常见低阶答案：“我会认真读”；“我会多找好词好句”；没有具体策略。

助学者提示：场面描写类文章要按“顺序 + 动作 + 画面 + 中心”读。

画像回收：回到 A8 的场面描写阅读策略。
