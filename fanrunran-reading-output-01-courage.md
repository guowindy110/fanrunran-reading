# 范润然自选素材第 1 篇训练题：《何为勇气》

> 版本：2026-06-11  
> 素材来源：`02石头《何为勇气》` MinerU 导出稿  
> 使用对象：范润然，五年级  
> 训练定位：不是泛泛谈“勇气”，而是训练她把故事、概念、例子和结论连成证据链。

---

## 1. 素材可用度结论

综合结论：**推荐使用，但要带支架。**

理由：

1. 文章有清楚的结构：小明和小刚的高墙故事 -> 妈妈解释勇气 -> 亚里士多德观点 -> 黄金中道与实践智慧 -> 小刚变化 -> 最终升华。
2. 文章有足够的原文证据，能训练“核心证据 vs 背景证据”。
3. 文章主题适合迁移，但迁移必须受文本约束，正好训练 A7。
4. 难点在于“黄金中道、实践智慧、怯懦、鲁莽”等抽象词，对五年级学生略难，需要助学者先帮她抓住故事主线。

本篇主目标轴：

- A4 篇章整合与概括
- A6 证据整合与论证支持

本篇辅助观察轴：

- A2 文本类型与文体识别
- A3 信息定位
- A7 评价判断与迁移
- A8 元认知监控与策略

不建议强行训练的轴：

- A1 不作为主训练。本文有术语，但不要把本篇变成字词解释课。
- A5 不作为主训练。人物理解可以顺带观察，但重点不是分析小明性格，而是理解“勇气”的概念边界。

阅读难度判断：**合适偏难。**

预计最容易暴露的卡点：

- 能复述小明帮助小刚，但说不清这个故事如何证明“勇气”。
- 能找出名言或妈妈的话，但不会解释证据作用。
- 迁移题容易写成“我要勇敢”“我要帮助同学”，缺少文本依据和边界。

是否适合进入本轮训练：**适合，建议作为第 1 篇 A4+A6 训练材料。**

## 4. 学生训练题

### 第 1 题：信息定位与文本结构

这篇文章不是只讲一个小故事。请按顺序写出文章的三个部分：

1. 先写了一个什么故事？
2. 中间借谁的观点解释“勇气”？
3. 最后用什么变化说明勇气产生了影响？

### 第 2 题：主线整合

请把文章中“小明对勇气的理解变化”补成一条链：

```text
看到高墙也害怕
-> ____________________
-> 听妈妈解释勇气
-> ____________________
-> 看到小刚后来主动挑战
-> 明白真正的勇气是：____________________
```

### 第 3 题：核心证据题

下面四条材料中，哪两条最能证明“真正的勇气不是无所畏惧，而是清醒、负责地面对困难”？哪一条只是背景证据？请说明理由。

A. 小明和小伙伴们在学校里关系很好。  
B. 小明自己也害怕高墙，但仍陪小刚一起爬，并一步一步鼓励他。  
C. 文中用“健康不只是没有生病”来类比勇气不能只理解为“不害怕”。  
D. 校园霸凌时，旁观是退缩，以暴制暴是鲁莽，客观告诉老师才是合理行动。

请这样答：

```text
我认为最核心的证据是____和____。
因为____证明了____，____证明了____。
____只是背景证据，因为它只能说明____，不能直接证明____。
```

### 第 4 题：评价迁移题

如果你看到同学被别人嘲笑，不敢参加班级展示，你怎样做才符合文中“真正的勇气”？请写出做法，并说明文中哪两处内容支持你的做法。

要求：

1. 不能只写“我要勇敢帮助他”。
2. 必须说明你的做法为什么不是退缩，也不是鲁莽。
3. 至少使用两处文章依据。

### 第 5 题：A8 复盘题，不计主分

做第 3 题时，你是怎么判断“核心证据”和“背景证据”的？

请补全：

```text
我先看题干要证明的是：____________________
我保留的证据是：____________________
我删掉的证据是：____________________
我删掉它的理由是：____________________
下次做证据题，我要先问自己：____________________
```

## 5. 出题人与助学者用答案要点

### 第 1 题

对应能力轴：A2 / A3

原文依据：高墙故事、亚里士多德观点、小刚后来再次攀爬。

满分答案要点：

1. 先写小明陪小刚攀爬高墙的故事。
2. 中间借妈妈和亚里士多德的观点解释勇气。
3. 最后用小刚后来主动挑战，说明勇气可以被传递，并引出全文结论。

常见低阶答案：

- 只写“讲小明和小刚爬墙”。
- 漏掉亚里士多德观点。
- 把最后一段环境描写当成主要内容。

助学者提示：

```text
你先不要概括全文，先给文章切三段：故事在哪里？讲道理从哪里开始？最后又回到谁身上？
```

评分点：3 分。结构三部分各 1 分。

画像回收：若只能复述故事，回收到 A2/A4 的“混合文本结构识别不足”。

### 第 2 题

对应能力轴：A4

原文依据：小明害怕但陪伴、小明听妈妈解释、亚里士多德观点、小刚后来的变化、结尾定义。

满分答案要点：

```text
看到高墙也害怕
-> 选择陪小刚一起面对
-> 听妈妈解释勇气
-> 理解勇气要有理性和分寸
-> 看到小刚后来主动挑战
-> 明白真正的勇气是在害怕时仍做清醒、负责、正确的选择
```

常见低阶答案：

- 写成“小明很勇敢，小刚也勇敢”。
- 只写事件，不写理解变化。
- 把“勇气就是不害怕”当成结论。

助学者提示：

```text
这题不是问发生了什么，而是问“理解怎么变深”。每个箭头后面都要写一个变化。
```

评分点：3 分。事件链 1 分，理解变化 1 分，最终定义 1 分。

画像回收：若能找事件但不能压成变化链，回收到 A4。

### 第 3 题

对应能力轴：A6

原文依据：小明害怕仍陪伴、校园霸凌例子、朋友关系背景、健康类比。

参考答案：

最核心的证据是 B 和 D。B 证明小明不是没有害怕，而是在害怕时仍选择陪伴和鼓励朋友；D 证明真正的勇气不是退缩，也不是冲动，而是根据情况做合理负责的选择。A 只是背景证据，因为它说明两人关系好，但不能直接证明什么是真正的勇气。C 是辅助说明，可以帮助理解“不能简单理解”，但不如 B 和 D 直接证明题干中的“清醒、负责地面对困难”。

常见低阶答案：

- 选 A，因为“朋友关系好所以有勇气”。
- 选 C 后只说“这个句子很有道理”，没有解释它证明什么。
- 把所有材料都当核心证据。

助学者提示：

```text
先看题干里的关键词：“不是无所畏惧”“清醒”“负责”。哪条材料直接说明这些词？
```

评分点：3 分。选出核心证据 1 分，说明证据作用 1 分，能区分背景或辅助证据 1 分。

画像回收：若只摘材料不解释“证明了什么”，回收到 A6。

### 第 4 题

对应能力轴：A7，可兼 A6

原文依据：小明陪伴小刚、校园霸凌例子、实践智慧关于对象/方式/后果的判断。

满分答案要点：

1. 做法要具体，例如陪同学准备、鼓励他一步步完成、必要时请老师创造安全支持。
2. 能说明这不是退缩，因为没有袖手旁观。
3. 能说明这不是鲁莽，因为没有起哄、硬推、责备或逼迫同学。
4. 能使用至少两处文本依据，例如小明陪伴小刚、实践智慧要考虑方式和后果、校园霸凌例子中的合理行动。

常见低阶答案：

- “我要勇敢帮助他”，但没有具体做法。
- “我让他马上上台”，忽略对方害怕，容易变成鲁莽。
- “我告诉他不要怕”，没有文本依据。

助学者提示：

```text
你的做法有没有同时避开两边：一边是不管他，一边是硬逼他？文中哪句话能支持这种分寸？
```

评分点：2 分。做法合理 1 分，文本依据与边界说明 1 分。

画像回收：若迁移只写常识，回收到 A7；若能找证据但不会解释，回收到 A6。

### 第 5 题

对应能力轴：A8

满分答案要点：

1. 能说出题干要证明的是“真正勇气的定义和边界”。
2. 能解释自己为什么保留 B/D，为什么删掉 A 或降低 C 的权重。
3. 能形成下次策略，如“先看题干关键词，再看证据是否直接证明它”。

常见低阶答案：

- “我凭感觉选的。”
- “哪个句子长就选哪个。”
- “我觉得都可以。”

助学者提示：

```text
你不是在选喜欢的句子，而是在选最能证明观点的句子。它能不能回答题干关键词？
```

评分点：1 分。能说清自己的判断方法即可。

画像回收：回收到 A8 的证据选择策略。

## 7. 助学者执行单

本篇训练主目标：A4 把故事和道理连成主线；A6 解释证据作用。

先不给提示时，先观察：

1. 她会不会把文章只当故事读。
2. 她能不能说出“妈妈的话”和“亚里士多德观点”在文章中的作用。
3. 她是否会把朋友关系、环境描写等背景内容当核心证据。

如果 A3 卡住，提示：

```text
先找三个位置：高墙故事在哪里？讲亚里士多德在哪里？小刚后来变化在哪里？
```

如果 A4 卡住，提示：

```text
你现在写的是事情顺序，还是“理解越来越深”的顺序？
```

如果 A6 卡住，提示：

```text
这条证据能不能直接证明“勇气不是不害怕，而是清醒负责”？不能直接证明的，先降为背景证据。
```

如果 A7 卡住，提示：

```text
你的做法有没有一边避免退缩，一边避免鲁莽？文中哪个例子讲了这个分寸？
```

如果 A8 复盘说不清，提示：

```text
你刚才删掉 A 的理由是什么？不是因为它错，而是因为它证明力不够直接。
```

本次可撤掉的支架：

- 如果她能独立完成第 1 题和第 2 题，下次可减少结构切段提示。

本次必须保留的支架：

- 第 3 题的“题干关键词 -> 证据是否直接证明”支架。
- 第 4 题的“退缩 / 鲁莽 / 合理负责”三分判断支架。

下一篇素材建议：

- 继续选“有一个故事 + 一个明确观点 + 至少两个例子”的材料。
- 不要马上换成纯抒情散文，先让 A6 证据判断稳定下来。
