# 范润然自选素材第 4 篇阅读题：《一粒麦子的旅行》

> 版本：2026-06-11  
> 素材来源：用户提供 Obsidian 文档《谁最中国-一粒麦子的旅行》  
> 使用对象：范润然，五年级  
> 训练定位：训练她读懂“空间路线 + 饮食文化例子 + 总结观点”的说明性散文，不把文章读成零散美食介绍。

---

## 1. 素材可用度结论

综合结论：**推荐使用，适合作为第 4 篇主线整合训练。**

说明：原始文档标题疑似出现 OCR/录入误差“妻子”，正文和文件名均指向“麦子”，本训练统一按《一粒麦子的旅行》处理。

理由：

1. 文章路线清楚：两河流域 -> 伊朗 -> 塔里木盆地/新疆 -> 河西走廊、关中平原、黄土高原 -> 淮中平原/阜阳。
2. 文章用不同面食说明小麦在不同土地上生长、流转、碰撞和交融，适合训练 A4 篇章整合。
3. 文章有多处具体证据：拉文什、桑嘎面包、馕、石子馍、阜阳馒头，适合训练 A6 证据作用。
4. 文章结尾从饮食写到文明与生活，适合做有限度的 A7 迁移。

本篇主目标轴：

- A4 篇章整合与概括
- A6 证据整合与论证支持

本篇辅助观察轴：

- A2 文本类型与文体识别
- A3 信息定位
- A7 评价判断与迁移
- A8 元认知监控与策略

不建议强行训练的轴：

- A1 不作为主训练。少量地名和食物名需要解释，但不把本篇变成词语课。
- A5 不作为主训练。本文不是人物理解材料。

阅读难度判断：**合适。**

预计最容易暴露的卡点：

- 只记住了很多食物名，说不清这些例子共同证明什么。
- 能复述路线，但不能概括“旅行”带来的文化交融。
- 迁移题容易写成“我喜欢吃面食”，没有回到文本中的文明与生活。

是否适合进入本轮训练：**适合。**

## 4. 学生训练题

### 第 1 题：路线定位题

请根据原文，按顺序写出小麦旅行中出现的几个关键地点或区域。

```text
两河流域 -> __________ -> 中国/塔里木盆地 -> __________ -> 关中平原 -> __________
```

再写出其中两个地方对应的面食。

### 第 2 题：篇章主线整合

这篇文章表面上写小麦旅行，实际上还写了什么？请把文章主线补完整：

```text
小麦从两河流域出发
-> 在不同地方变成不同的面食
-> 这些面食进入当地人的__________
-> 小麦改变了中国和世界的__________
-> 作者最后想说明：小麦把__________串联起来
```

### 第 3 题：核心证据题

下面四条材料中，哪两条最能证明“麦子不仅改变饮食，也进入了人的生活和文化”？哪一条主要是路线背景，不能直接证明文化意义？请说明理由。

A. 麦子从两河流域出发，翻过高原来到中国，又到达欧洲和北非。  
B. 馕在维吾尔文化里有平安、团圆、吉祥之意，远行的人吃一口家里的馕能够减轻乡愁。  
C. 阜阳馒头蘸酱豆，是阜阳人舌尖上最难忘的记忆。  
D. 烤馕需要和面、配料、造型、预热馕坑、打馕等多个步骤。

请这样答：

```text
我认为最核心的证据是____和____。
因为____证明了____，____证明了____。
____主要是路线或制作背景，不能直接证明文化意义，因为____。
```

### 第 4 题：评价迁移题

如果有同学说：“这篇文章只是介绍几种面食，没有更深的意思。”你同意吗？请根据原文回答。

要求：

1. 先表明同意或不同意。
2. 至少使用两处原文证据。
3. 说明这些证据怎样证明“小麦的旅行”和人类生活、文化有关。

### 第 5 题：A8 复盘题，不计主分

这篇文章地名和食物名很多。你做题时是怎么防止自己被名字带跑的？

```text
我先抓住的路线是：____________________
我把例子分成了：____________________
我认为最能证明中心的例子是：____________________
因为它证明了：____________________
下次读这种文章，我要先问自己：____________________
```

## 5. 出题人与助学者用答案要点

### 第 1 题

对应能力轴：A3

答案要点：两河流域 -> 伊朗/伊朗高原 -> 中国/塔里木盆地/新疆 -> 河西走廊/黄土高原 -> 关中平原 -> 淮中平原/阜阳。面食可写拉文什、桑嘎面包、馕、石子馍、阜阳馒头。

常见低阶答案：只列食物不列路线；把欧洲、北非和中国路线混在一起；漏掉“中国境内继续向东”的路线。

助学者提示：先找地名，再找地名旁边出现的食物名。

评分点：3 分。路线基本顺序 2 分，地点与食物对应 1 分。

画像回收：若定位混乱，回收到 A3。

### 第 2 题

对应能力轴：A4

参考答案：

```text
小麦从两河流域出发
-> 在不同地方变成不同的面食
-> 这些面食进入当地人的生活和情感
-> 小麦改变了中国和世界的饮食格局
-> 作者最后想说明：小麦把人类的生命与生活串联起来
```

常见低阶答案：只写“小麦去了很多地方”；只写面食清单；不能从食物上升到生活和文明。

助学者提示：这题不是问“有哪些面食”，而是问“这些面食共同说明什么”。

评分点：3 分。路线 1 分，生活文化 1 分，中心提升 1 分。

画像回收：若能找信息但不能概括主线，回收到 A4。

### 第 3 题

对应能力轴：A6

参考答案：最核心的证据是 B 和 C。B 说明馕不只是食物，还承载平安、团圆、吉祥和乡愁；C 说明阜阳馒头进入了人的味觉记忆和生活经验。A 主要说明小麦旅行路线，是背景主线，但不能直接证明文化意义。D 主要说明制作步骤，可以辅助说明食物做法复杂，但比 B、C 更不能直接证明“生活和文化”。

常见低阶答案：选 A，因为它最长、最像主线；选 D，因为步骤多；只摘句子不解释证据作用。

助学者提示：题干问“生活和文化”，能证明情感、记忆、吉祥意义的证据更核心。

评分点：3 分。选出核心证据 1 分，解释证明作用 1 分，能区分路线/制作背景 1 分。

画像回收：若只看信息量不看证明力，回收到 A6。

### 第 4 题

对应能力轴：A7，可兼 A6

参考答案方向：不同意。文章虽然写了拉文什、桑嘎面包、馕、石子馍、阜阳馒头等面食，但这些例子不是单纯介绍吃的。比如馕在维吾尔文化里有平安、团圆、吉祥之意，远行的人吃馕能减轻乡愁，说明食物承载情感和文化。阜阳馒头和酱豆是阜阳人舌尖上最难忘的记忆，说明小麦进入了人的生活记忆。结尾还说小麦奠定“南稻北麦”的饮食格局，串联人类生命与生活，所以文章的深层意思是小麦旅行影响了饮食、文化和文明。

常见低阶答案：只写“不同意，因为面食很好吃”；只写自己喜欢的食物；没有使用原文证据。

助学者提示：回答“有没有更深意思”，要回到结尾和文化证据，不要只谈个人口味。

评分点：2 分。观点明确 1 分，两处证据能说明文化/生活意义 1 分。

画像回收：迁移空泛回收到 A7；有证据但解释弱回收到 A6。

### 第 5 题

对应能力轴：A8

满分答案要点：能说出先抓路线，再把例子分为“地域面食、文化意义、生活记忆、中心总结”；能说明最核心例子不是名字最特别的，而是最能证明中心的。

常见低阶答案：“我把面食都背下来”；“我喜欢馕所以选馕”；说不清为什么某条证据更核心。

助学者提示：地名和食物名是材料，不是答案。先问“它证明了什么”。

评分点：1 分。能说出路线管理和例子分类策略即可。

画像回收：回收到 A8 的说明文信息管理策略。

